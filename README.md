# Task Manager CI/CD

## Overview
This project is a Task Manager application designed to help users manage their tasks efficiently. It includes features for adding, updating, and deleting tasks, as well as tracking progress.

## Features
- User authentication
- Create, read, update, and delete tasks
- Task categorization
- Progress tracking

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/jassisamuran/task-manager-cicd.git
   ```
2. Navigate to the project directory:
   ```bash
   cd task-manager-cicd
   ```
3. Install the required dependencies:
   ```bash
   npm install
   ```

## Usage
To start the application, run:
```bash
npm start
```
